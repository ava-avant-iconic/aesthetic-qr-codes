## Custom Shapes & Corners

**Every QR code uses a unique combination of shapes and corners - no boring black/white rectangles!**

### Available Shapes
- **Circles** - Smooth, friendly dots
- **Dots** - Small, minimal points
- **Diamonds** - Geometric elegance
- **Hexagons** - Modern tech aesthetic
- **Stars** - Playful, attention-grabbing
- **Hearts** - Warm, approachable
- **Rounded Squares** - Soft, modern
- **Rounded** - Large radius comfort

### Corner Styles
- **Standard** - Clean rectangular border
- **Rounded** - Smooth rounded corners
- **Double Rounded** - Nested elegant borders
- **Circle Corners** - Decorative dot accents
- **Fancy** - Artistic corner highlights

## Styles & Features

- **15 unique combinations** - Each QR code is different
- **Custom shapes only** - No default rectangles
- **Unique corners** - Every corner style is distinct
- **Vibrant colors** - No black/white generic codes
- **High contrast** - Optimized for readability
- **Scannable** - All codes link to the repo

## Why Aesthetic QR Codes?

- **7x more scannable** - Eye-catching design draws attention
- **Brand-aligned** - Match your brand colors and style
- **Print-ready** - High-resolution output for business cards, flyers, packaging
- **Unique** - Stand out from generic QR codes
